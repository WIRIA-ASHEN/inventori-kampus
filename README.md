# Sistem Inventori Kampus

Aplikasi berbasis Laravel untuk mengelola data inventaris kampus.

## Fitur

- Login
- Dashboard
- Manajemen Barang
- Manajemen Kategori
- Manajemen Ruangan
- Pinjam Barang
- Keluhan Barang

## Teknologi

- Laravel
- PHP
- MySQL
- Bootstrap
